# Untyped λ-calculus

- **[md/LC/index](md/LC/index/)** is the root of a section of this website showing
a lightweight Agda formalization of the denotational semantics of the untyped λ-calulus.

- **[md/Library/index](md/Library/index/)** is the root of a section of this website showing
Agda modules imported for use in some formalization.

- **[html/LC.index](html/LC.index.html)** is the root of the orginal HTML pages
generated by Agda from (potentially literate) source code in the GitHub repository.

- **[pdf/ULC.All](pdf/LC.index.pdf)** is a highlighted PDF listing of all the Agda modules.
